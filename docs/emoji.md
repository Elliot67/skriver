# Slack

Open Slack in a browser, the list is available in a JS script. Make sure to retrieve the largest list.
Translations are loaded separately, we don't need them.

# Microsoft Teams

Open Teams in a browser without any cache, the list is available on the load in a json file.
Each language has a file, make sure to use the en-US version.

# Jira

Full list can be retrieve from [a single URL](https://ecosystem.atlassian.net/gateway/api/emoji/standard)

# Notion

Open Notion in a browser, the list is available in a JS file `_assets/emojiData-3f1ba54fa6ae8074.js`, then use `emojiByName` only.
